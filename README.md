## Setup

#### Prerequisites
* qtile-extras

   ```sh
   sudo pacman -S qtile-extras
   ```

#### Installation 
1. Clone the repo

   ```sh
   git clone https://github.com/Tardz/Laptop.git
   ```
   
2. Run install script located at Laptop/install.sh
   
   ```sh
   ./install.sh
   ```
   
3. Restart Qtile
   
   ```sh
   qtile cmd-obj -o cmd -f restart
   ```
