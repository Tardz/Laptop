### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Tardz/Laptop.git
   ```
3. Run install script located at Laptop/install.sh
   ```sh
   ./install.sh
   ```
4. Restart Qtile
   ```sh
   qtile cmd-obj -o cmd -f restart
   ```
